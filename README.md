## Description

A tasks API to explore the Nest.js framework [Nest](https://github.com/nestjs/nest).
Nest.js is a Node.js framework that uses TypeScript.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
